# shinkai-manga-skill

一个用于生成新海诚风格 AI 漫剧的 Claude Skill。

## 功能

输入故事方向或剧本，输出：
- 角色锁定档案（保证跨镜头一致性）
- 新海诚风格分镜脚本（含构图/色调/时长/旁白）
- 可直接粘贴进 Seedance / 小云雀的视频生成 prompt
- 生成后自检清单

## 使用方式

安装 `.skill` 文件后，在 Claude 对话中直接描述你的故事方向即可触发。

支持三种输入状态：
- 只有氛围/情绪词 → Claude 帮你建立世界观
- 有角色没情节 → Claude 帮你补全冲突结构
- 有完整剧本 → 直接生成分镜和 prompt

## 风格系统

内置新海诚美学规则库：色彩哲学、构图法则、叙事节奏、标志性元素。
同时内置爆款结构模块，区分哪些平台传播逻辑与新海诚美学兼容、哪些会破坏风格。

## 适配工具

Seedance 2.0 / 小云雀 / 即梦
